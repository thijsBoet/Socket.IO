# Socket.IO
----

## Before Socket.IO
* Socket.IO build upon **Web Socket** API, which is build upon TCP/IP

#### Network Layers
* App - HTTP, FTP, SSH | Node/Express, Flask
* Transport - TCP/UDP  | Websocket / Socket.IO
* Network - IP         | Websocket / Socket.IO
* Link - WIFI/Ethernet
* Physical - Cable
